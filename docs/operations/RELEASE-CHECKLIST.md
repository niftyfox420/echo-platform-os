# Release Checklist

- Confirm the working version number.
- Update `CHANGELOG.md`.
- Run PHP syntax validation.
- Build the installable ZIP.
- Test replacement installation on staging.
- Confirm activation without a fatal error.
- Confirm Mission Control and critical modules open.
- Run a catalog health scan.
- Confirm no duplicate Echo admin menus.
- Keep the previous working ZIP available for rollback.
